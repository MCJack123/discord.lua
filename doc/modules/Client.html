initialize (options)
Internally initialize's the Client class. Please use Client:new(options) instead.

Parameters:

  • options Options table (Currently useless.)
login (email, password)
Logs the Client into Discord's servers. This is required to use most of the Clients functions.

Parameters:

  • email E-mail Address of the account to log in.
  • password Password of the account to log in. (WARNING: Do NOT store this password in a GitHub repo or anything of the sorts.)

Returns:

    True or False depending on success.
logout (email, password)
Logs the Client out of the Discord server. (This is currently useless/does nothing)

Parameters:

  • email E-mail Address of the account to log in.
  • password Password of the account to log in. (WARNING: Do NOT store this password in a GitHub repo or anything of the sorts.)

Returns:

    True or False depending on success.
